Waterfall

One cascade routes the whole market

Each practitioner is tested top-to-bottom (uses CGM, then metabolic-adjacent, then everyone else) and routes to exactly one campaign. Build the strategy once, then keep adding leads.

first match wins 
1. IF a CGM mention is found, or they sell a glucose / metabolic program -> Tier 1 (Believer in the modality, but building it by hand)
2. IF ELSE they position on weight, blood sugar, pre-diabetes, longevity or GLP-1 -> Tier 2 (Adjacent and ripe to add the CGM modality)
3. IF ELSE any remaining practitioner who serves individuals -> Catch-all (Clients want visible results; the practitioner wants retention)

Launch

Prove the hottest tier, then widen

Start with practitioners already using CGMs, then the metabolic-adjacent pool, then the catch-all, running the GLP-1 side campaign in parallel.

launch first
01

Stand up sending infrastructure, warmup and inboxes on burner domains separate from viiraa.com.

02

Run the LinkedIn, directory, Google Maps and bio-scrape pulls for the launch market. Build and validate the routing matrix.

03

Launch Tier 1 (CGM-active) first. Smallest market, highest conviction, fastest proof of the done-for-you-product thesis.

04

Add Tier 2 (metabolic-adjacent) once it clears 1,500 contacts, keeping Tier 1 running. The biggest adjacent-and-ready pool.

05

Roll the Tier 3 catch-all once the sharper tiers have validated copy winners to borrow from. Add leads continuously; the routing logic never changes.

06

Run the GLP-1 clinic side campaign in parallel once Tier 1 is live. Layer corporate wellness, OTC-CGM and creator lanes only after the core waterfall is stable.

Scale / cut rules

1 positive reply / 150 contacts Perfect. Scale this tier hard.
1 / 300 Good. Let it run and add leads steadily.
1 / 450-500 Borderline. Test a copy or lead-magnet variant.
Worse than 1 / 500 or <1% after 1,500 sends Cut the tier and rework the angle.
