Waterfall

One cascade routes the whole market

Every checkout-owning platform flows top to bottom: competitor widget, monetization hire, refund pain, payment rails, or catch-all.

first match wins 
1. IF owns_checkout != true -> Excluded (No checkout, no attach point)
2. IF insurance_widget_detected != none -> Tier 1 (Competitor takedown)
3. IF monetization_hire = true -> Tier 2 (Active intent)
4. IF refund_policy_language = strict -> Tier 3 (Refund leakage)
5. IF payfac_detected != none -> Tier 4 (Rails already built)
6. IF remaining checkout-owning platform with headcount >= 11 -> Catch-all (Owns the transaction)
