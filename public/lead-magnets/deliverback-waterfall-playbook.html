Waterfall

One strategy, whole market

A priority-routed cascade covers 100% of the market with no property left behind.

first match wins 
1. IF A lost-and-found tool is detected (Chargerback, NotLost, Crowdfind, ReturnMe, iLost) -> Tier 1 - Competitor takedown (Already sold on the category, but paying for software that only matches items.)
2. IF Else, a lost-property complaint is visible in reviews -> Tier 2 - Reputation rescue (A guest's lost item became a public 1-2 star review, hurting bookings right now.)
3. IF Else, 200+ reviews or a large / XL property -> Tier 3 - Volume / staff-time (High guest throughput means high left-behind volume burning front-desk hours.)
4. IF Else (everything remaining) -> Tier 4 - Catch-all (Every property runs lost-and-found ad hoc, with no system and zero upside captured.)
