Waterfall

One cascade routes every WA investor

Portfolio size plus acquisition recency creates exclusive tiers, from repeat buyers to scaled dormant portfolios to emerging investors and catch-all entities.

first match wins 
1. IF purchases_trailing_12mo >= 2 -> Tier 1 (In-Market Repeat Buyer)
2. IF properties_owned_count >= 3 -> Tier 2 (Scaled Portfolio, Dormant)
3. IF properties_owned_count in [1,2] AND owner_occupied == false -> Tier 3 (Emerging Investor)
4. IF all remaining WA investor entities -> Catch-all (Remaining RE LLCs, cash buyers, absentee owners)
