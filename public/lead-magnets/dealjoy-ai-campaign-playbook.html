Waterfall

One cascade routes every producing agent

Every agent is tested top-to-bottom by proven listing intent and routes to exactly one seller-farming campaign.

first match wins 
1. IF runs_seller_ads = true OR seller_capture_embed = true -> Tier 1 (Paid seller-lead spenders)
2. IF sales_last_12mo >= 8 AND seller_side_share = low AND active_listings <= 1 -> Tier 2 (Buyer-dependent producers)
3. IF farm_concentration = '1-2 ZIPs' AND tenure_years >= 4 -> Tier 3 (Established manual farmers)
4. IF every remaining licensed, producing agent in the ZIP -> Tier 4 (Exclusivity / scarcity catch-all)
