Waterfall

How the cascade works

Exclusive routing, priority order, and a catch-all that sweeps the whole market.

first match wins 
1. IF booking_tool = Mindbody -> Tier 1 — Mindbody takedown (The most-switched-from incumbent and Walla's explicitly named competitor: highest pain, sharpest intent.)
2. IF ELSE IF tool_class = generic-scheduler (Vagaro, Square, Acuity, Calendly, Wix, Booker, Fresha) -> Tier 2 — outgrown a generic tool (A general scheduler was never built for a studio; memberships, payroll, and retention are done by hand around it.)
3. IF ELSE IF tool_class = modern-boutique (Glofox, Mariana Tek, Momence, Arketa, Zen Planner, Pike13, WellnessLiving) -> Tier 3 — modern-tool gap (Already on a real platform, but paying for gaps: weak retention AI, thin support, or per-location cost at scale.)
4. IF ELSE (no booking widget detected — phone / email / DM / Google Form) -> Tier 4 — catch-all, no system (Running the studio manually; the first-real-system pitch. Sweeps the rest of the market so coverage is 100%.)
