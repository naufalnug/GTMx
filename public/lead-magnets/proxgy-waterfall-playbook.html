02

The waterfall

Every account tested top-to-bottom, routed to the first tier it matches.

first match wins 
1. IF industrial_field_workforce is FALSE -> Not in market (No hazardous vertical or field workforce means no safety gap to close. Not in the TAM.)
2. IF recent_incident is not none (last ~12 months) -> Tier 1, Prevent the Next One (An incident just happened, so there is acute, board-level, time-boxed pressure to prove it won't recur.)
3. IF else, hiring_hse_role is TRUE -> Tier 2, Equip the Team You're Building (Safety budget is being spent now, but on headcount, one officer still can't watch every worker.)
4. IF else, published_safety_kpi is not none -> Tier 3, Your KPI Is Manual and Lagging (They measure safety and answer to a board on it, but the number is compiled by hand, weeks late.)
5. IF else, site_count is 5 or more -> Tier 4, You Can't Be at Every Site (Crews spread across sites means no inspector can be everywhere, so hazards surface late and travel bleeds.)
6. IF else (field workforce, no sharper signal) -> Catch-all, Blue-Collar Has No Tools (The structural gap Proxgy was built for. Broadest, softest, catches everything above missed.)

05

06

Scale / cut rules

1 positive reply per 150 contacts Perfect. Scale the tier hard.
1 per 300 Good. Let it run untouched.
1 per 450-500 Borderline. Test one copy variant, vary the lead magnet first.
Worse than 1 per 500, or reply rate under 1% after 1,500+ sends Dead. Cut and rework, don't tinker.
