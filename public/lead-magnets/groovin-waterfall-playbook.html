Waterfall

One route, first match wins

Every account tests top-down and routes to the first tier it matches. Later tiers are explicitly 'not the tier above,' so nothing double-matches.

first match wins 
1. IF competitor_tool_detected != 'none' -> TIER 1 (Competitor takedown, strongest intent)
2. IF affinity_detected in {confirmed, inferred-high} -> TIER 2 (Affinity / VC, best ACV and retention)
3. IF attio_detected in {confirmed, inferred} -> TIER 3 (Attio / B2B sales, medium-high)
4. IF no tool detected but ICP-fit -> TIER 4 (Catch-all soft pitch, 100% coverage)

1 positive reply / 150 contacts Perfect. Scale this tier hard.
1 / 300 Good. Let it run.
