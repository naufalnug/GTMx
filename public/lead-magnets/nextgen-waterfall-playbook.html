Waterfall

One cascade routes the entire TAM

Every account exits with exactly one tier. Strongest hiring intent wins first, and the catch-all keeps coverage at 100%.

first match wins 
1. IF open_automatable_reqs >= 4 OR team_scaling_language = true -> Tier 1 (Team Build-Out)
2. IF open_automatable_reqs BETWEEN 2 AND 3 -> Tier 2 (Multi-Function)
3. IF open_automatable_reqs = 1 -> Tier 3 (Single Seat)
4. IF open_automatable_reqs = 0 AND point_ai_tool_detected != none -> Tier 4 (Orphaned AI / No Command Center)
5. IF in TAM, no reqs, no AI tool -> Catch-all (Future-Proof)
