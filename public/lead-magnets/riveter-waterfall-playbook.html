Waterfall

One cascade routes the whole market

Every account flows top to bottom: running a scraper stack, hiring to build, a data-dependent product, or the catch-all. Most land in exactly one campaign.

first match wins 
1. IF scraping_tool_detected != none OR eng_scraping_signal != none -> Tier 1 (Already running a scraper stack)
2. IF scraping_job_open = true AND posted within 90 days -> Tier 2 (Hiring to build it in-house)
3. IF data_product_type != none -> Tier 3 (Data-dependent product company)
4. IF any remaining B2B team with a research-heavy function -> Catch-all (Doing web research by hand)
