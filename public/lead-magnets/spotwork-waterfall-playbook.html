Waterfall

One cascade routes the whole industrial TAM

Every facility is tested top-to-bottom by intent: competitor takedown, acute fill gap, variable demand, then the bench-warmup catch-all.

first match wins 
1. IF gig_competitor_detected != none (Instawork / Traba / Veryable / Bluecrew / Wonolo) -> Tier 1 (Competitor takedown)
2. IF open_hourly_reqs >= 5 OR req_urgency = urgent -> Tier 2 (Acute fill gap)
3. IF open_hourly_reqs between 1 and 4 OR seasonal_language = true -> Tier 3 (Variable / seasonal demand)
4. IF every remaining industrial facility in a Spotwork metro -> Catch-all (Warm the bench)
