Waterfall

One cascade routes every manufacturer in the market

Each owner is tested top-to-bottom by intent: on-market, succession-vulnerable, aging owner, then the relationship catch-all.

first match wins 
1. IF on_market_listing != none OR retirement_language = true -> Tier 1 (On-market / decided)
2. IF NOT Tier 1 AND owner_operated AND year_established <= 25yrs ago AND owner_est_age >= 60 AND successor_present = none -> Tier 2 (Succession-vulnerable)
3. IF NOT Tier 1-2 AND owner_operated AND owner_est_age >= 55 -> Tier 3 (Aging owner)
4. IF every remaining owner-operated manufacturer in the footprint -> Catch-all (Relationship)
