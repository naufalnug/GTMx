02

The Waterfall

Every lead is tested top to bottom and routes to the first tier it matches.

first match wins 
1. IF A live SDR/BDR/BD role is posted -> Tier 1 - Hiring headcount (Strongest intent: they have already decided to invest in outbound.)
2. IF Else, cold-email infra or a tool is detected -> Tier 2 - Running cold email (They are doing outbound the hard way and can be taken over.)
3. IF Else, active paid ads are detected -> Tier 3 - Renting via ads (They pay for leads that stop when spend stops.)
4. IF Else, any reachable B2B service firm -> Catch-all - Referral only (Broadest reach; no repeatable channel yet.)

The Launch

Phase-roll the tiers so each launches with proof behind it.

launch first
01

Launch Tier 1 first: highest intent, fastest proof, cleanest case study (Spark Talent's 10-15 opps/month directly answers hire-vs-system).

02

Phase in Tier 2 (the takedown) once Tier 1 shows 1/300 or better - the free deliverability audit is a low-friction hook.

03

Add Tier 3 and the Catch-all as the scrapes deliver 1,500+ contacts each.

04

Run side campaigns (fresh funding, new growth-leader hire, conference follow-up) only after the core waterfall is stable.

Scale / cut rules

1 positive reply / 150 Perfect - scale the tier
1 / 300 Good - let it run
1 / 450-500 Borderline - test a copy or lead-magnet variant
Above 1 / 500, or <1% reply after 1,500+ sends Cut and rework
