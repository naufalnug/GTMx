The Waterfall

One cascade, whole market

Each account is tested top-to-bottom and routes to the first tier it matches. Built once, mapped to the full TAM, run for 11+ months.

first match wins 
1. IF A competing journey tool is detected -> Tier 1 - Competitor Takedown (They bought the category. Budget exists and they are feeling the mapping ceiling.)
2. IF Else, a senior CX leadership title exists -> Tier 2 - Mandate, No System (The mandate is real but journeys live in slides. This is the decision gap, named.)
3. IF Else, actively hiring a CX role now -> Tier 3 - Scaling CX (Team is growing. Stand up the operating system before spreadsheets harden.)
4. IF Else (any enterprise CX function) -> Catch-all - The Decision Gap (Has data and maps but no systematic way to decide what to fix or prove it worked.)
