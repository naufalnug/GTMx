Waterfall

One cascade routes the whole market

Each company is tested top-to-bottom by role type and routes to exactly one campaign. Build the strategy once, then keep adding leads.

first match wins 
1. IF hiring AI / automation / RevOps / GTM engineer -> Tier 1 (They've decided automation is worth a salary)
2. IF ELSE hiring SDR / BDR / appointment-setter -> Tier 2 (Manual prospecting and booking)
3. IF ELSE hiring support / CX / receptionist -> Tier 3 (Inbound volume outpacing staff)
4. IF ELSE hiring data-entry / admin / VA / ops coordinator -> Tier 4 (Humans doing repetitive back-office work)
5. IF ELSE every remaining company with no matching open role -> Catch-all (Greenfield, quiet manual-work loss)

Tiers

Each open role maps to a different agent

An open SDR req gets the AI SDR pitch, a support req gets the 24/7 voice agent, an automation req gets buy-vs-hire. Pain is inferred from the trigger, never invented.

Launch

Start with companies hiring the automation role itself, then sales, then support, then ops and the catch-all.

launch first
01

Stand up sending infrastructure, warmup and inboxes on burner domains separate from venaflare.com.

02

Pull LinkedIn Jobs, Clay/Apollo job-postings and careers-page data for the launch market. Build and validate the routing matrix.

03

Launch Tier 1 (automation hires) plus the Creative Ideas broad campaign first. Smallest, hottest, fastest proof of the buy-vs-hire thesis.

04

Add Tier 2 (sales hires) once it clears 1,500 contacts, keeping Tier 1 running. This is the flagship offer.

05

Roll Tier 3 (support), then Tier 4 and the catch-all once top tiers are calibrated and copy winners exist. Add leads continuously; the routing logic never changes.

06

Layer funding, new-leader, platform-but-manual and inbound-leak side campaigns only after the core waterfall is stable.

Scale / cut rules

1 positive reply / 150 contacts Perfect. Scale this tier hard.
1 / 300 Good. Let it run and add leads steadily.
1 / 450-500 Borderline. Test a copy or lead-magnet variant.
Worse than 1 / 500 or <1% after 1,500 sends Cut the tier and rework the angle.
