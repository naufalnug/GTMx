Waterfall

One cascade routes the whole market

Every restaurant flows top to bottom: already sells plant-based meat, has vegan cheese but no protein, high-fit cuisine, or catch-all.

first match wins 
1. IF vegan_meat_on_menu != none -> Tier 1 (Switch: category already proven)
2. IF vegan_cheese_or_section = true AND plant_based_meat_present = false -> Tier 2 (Fill the gap: demand present, protein missing)
3. IF cuisine_type in pizzeria, deli-sandwich, brewery-gastropub -> Tier 3 (High-fit, cold: clean drop-in cuisine)
4. IF any remaining foodservice operator -> Catch-all (Sample-first: plant-based demand growing regardless)

Scale / cut rules

1 positive reply / 150 contacts Excellent. Scale the tier hard.
1 / 300 Good. Let it run.
1 / 450-500 Borderline. Test one copy or lead-magnet variant.
Worse than 1 / 500 or <1% after 1,500 sends Cut the tier or rework it from the trigger up.
