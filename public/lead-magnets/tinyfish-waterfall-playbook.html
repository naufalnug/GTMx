Waterfall

One cascade routes the whole market

Each company is tested top-to-bottom (tool, then hiring, then vertical) and routes to exactly one campaign. Build the strategy once, then keep adding leads.

first match wins 
1. IF a web-data / scraping / browser-automation tool is detected -> Tier 1 (Proven pain and budget for live web data)
2. IF ELSE an open role mentions scraping / crawling / web automation -> Tier 2 (Building brittle scrapers in-house)
3. IF ELSE the business model is a live-web-data vertical -> Tier 3 (Pricing and inventory decisions run on stale data)
4. IF ELSE every remaining company building agents or products -> Catch-all (Agents hit reliability ceilings on cached web results)
