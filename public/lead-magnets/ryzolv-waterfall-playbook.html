Waterfall

One cascade routes the whole regulated market

Every regulated enterprise flows top to bottom: governance hire, production hire, build hire, or catch-all. First match wins.

first match wins 
1. IF industry not in finance, healthcare, pharma, manufacturing (or adjacent regulated) -> Excluded (No audit obligation, no wedge)
2. IF ai_role_hiring = governance / responsible-AI / model-risk -> Tier 1 (Already at the governance wall)
3. IF ai_role_hiring = production / MLOps / ML platform -> Tier 2 (Pilot stuck in POC purgatory)
4. IF ai_role_hiring = build / ML / AI / LLM engineer -> Tier 3 (Building without governance in the room)
5. IF regulated enterprise, 500+, no AI hiring visible -> Catch-all (On the roadmap, not yet started)
