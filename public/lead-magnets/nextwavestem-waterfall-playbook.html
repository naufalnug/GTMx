Waterfall

How leads route

Each account is tested top to bottom and lands in the first tier that fits, so most get exactly one relevant sequence.

first match wins 
1. IF Already a NextWave customer -> Suppress (Never cold-email an existing or approved-vendor district.)
2. IF A live STEM/CTE/CS/robotics teacher opening -> Tier 1 - Staffing Gap (Budget plus an urgent, provable gap right now.)
3. IF Mandate state OR funding marketplace, AND no STEM program page -> Tier 2 - Mandate + Money (A deadline and money, but no vendor chosen yet.)
4. IF A single, one-off STEM program detected -> Tier 3 - Program Upgrade (They value STEM but have no full K-12 pathway.)
5. IF Everything else -> Tier 4 - Access & Equity (Catch-all guarantees 100% market coverage.)

Scale / cut rules

1 positive reply per 150 Excellent - scale this tier aggressively.
1 per 300 Good - let it keep running.
1 per 450-500 Borderline - test a new copy variant.
Worse than 1 per 500 Underperforming - rework or cut the tier.
